Порядок установки WSL2:
1. Запустить Install1.bat от имени администратора
2. Установить Ubuntu и пакет обновления ядра WSL2 с открывшихся сайтов
3. Открыть Ubuntu и создать пользователя (ввести логин и пароль)
3. Запустить Install2.bat НЕ от имени администратора
4. Запустить Run.bat
5. В случае, если не удается подключиться к удаленному рабочему столу, необходимо запустить Ubuntu
от имени администратора, вручную выполнить в нем следующую команду и затем снова
запустить Run.bat:
sudo /etc/init.d/xrdp start
